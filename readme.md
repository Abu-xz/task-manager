🛠️ Setup Instructions

1. Clone the repo

git clone: https://github.com/Abu-xz/task-manager.git

cd task-manager-api

2. Install dependencies
   npm install

3. Run in development mode
   npm run dev

4. Build TypeScript to JavaScript
   npm run build

5. Start production server
   npm start
